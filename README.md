# MorrisJS-Test
Ejemplo de uso de Morris JS con php para crear gráficos. 

---

Herramientas usadas:

```json
"dependencies": {
  "morris.js": "~0.5.1",
  "bootstrap": "~3.3.6"
}
```

Php, mysql para obtener los datos y poder generar la data que usan los gráficos. :smile:
